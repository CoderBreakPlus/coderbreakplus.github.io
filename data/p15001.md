# P15001 摧毁时间线 题解

## 解题思路

### 1. 简化问题：去掉第三项代价
首先，我们考虑如果去掉能量计算式子中的第三项（即假设不需要考虑 Future 控制器的位置），这是一个经典的区间 DP 问题。
我们可以设 $f_{l,r}$ 表示区间 $[l,r]$ 内部的时刻全部被删除了，且 $l-1$ 和 $r+1$ 没被删除时的最大能量贡献。这个状态可以在 $O(n^3)$ 的时间复杂度内完成转移。

### 2. 原问题：加上第三项代价（延后处理 Trick）
现在我们需要考虑加上式子的第三项 $(a_{id_{i+1}} - z_{id_{i+2}})^2$ 怎么做。

由于每次删除一个点时，它右侧相连的“下一个时刻”可能会受到之前操作的影响而改变，我们可以使用**延后处理 (Delayed Evaluation)** 的技巧。
我们在 DP 状态中额外记录“右侧被借用的点”的信息。

具体地，设 $f_{l,r,i,j}$ 表示：
- 目前区间 $[l,r]$ 被删了，且边界 $l-1$ 和 $r+1$ 没被删；
- 右侧用到了之后的点 $i$（即删除操作右侧碰到的点是 $i$）；
- 限制区间 $[l,r]$ 内的点 $j$ 必须要在 $[l,j)$ 全部被删了之后，才能被删除；
在满足以上条件下的贡献最大值。

### 3. 状态转移方程
我们考虑枚举区间 $[l,r]$ 中**最后删除的点 $k$**。
由于有上述关于 $j$ 的限制，显然必须满足 $j \le k$。
接着，我们要枚举左侧区间 $[l, k-1]$ 被删除时，要“借用”到的右侧点 $x$（这个 $x$ 一定在区间 $[k+1, r]$ 内部）。

由此可以得到第一种转移方程：
$$f_{l,r,i,j} \leftarrow f_{l,k-1,x,j} + f_{k+1,r,i,x} + (a_{l-1}-x_k)^2 + (a_k-y_{r+1})^2 + (a_{r+1}-z_i)^2$$

特别地，我们还需要考虑 $j=r+1$ 的情况，此时表示**不限制** $j$ 在区间内的删除顺序。
对于这种情况，有第二种转移：
$$f_{l,r,i,r+1} \leftarrow f_{l,k-1,x,k} + f_{k+1,r,i,x} + (a_{l-1}-x_k)^2 + (a_k-y_{r+1})^2 + (a_{r+1}-z_i)^2$$

### 4. 复杂度
状态数为 $O(n^4)$，转移时需要枚举 $k$ 和 $x$，转移复杂度 $O(n^2)$。
因此总时间复杂度为 $O(n^6)$。由于常数较小（有很多非法的区间和状态会被直接剪枝），可以通过此题。

## 深入探讨：为什么转移时能够共享同一个 $i$？

**疑问：** 
在状态转移 $f_{l,r,i,j} \leftarrow f_{k+1,r,i,x} + \dots$ 中，为什么区间 $[l,r]$ 转移到右边 $[k+1,r]$ 的时候，用到的之后的点 $i$ 和 $[l,r]$ 整体用到的后边点**一样**？
明明在物理上，我们可以“交错删除”——先删除 $[k+1,r]$ 中的最后一个元素，然后去在 $r+1$ 之后的地方删除一些元素（导致右侧点改变），最后再回来删除 $k$。这样它们用到的后边点就不一样了啊？

**解答：**
物理上确实允许交错删除，但在求**最大能量（最大化）**的背景下，交错删除的收益**永远不可能大于**集中删除的收益。因此，DP 只需要考虑不交错的情况就能找到全局最优解。

具体的数学证明如下：

**1. 代价公式的独立性与常数底数**
当我们在删除 $[l, r]$ 区间内的元素（包括 $k$ 以及 $[k+1, r]$ 内的元素）时，只要它们被删除时右侧碰到了边界 $r+1$，那么删除它所消耗能量的第三项固定形式为：
$$(a_{r+1} - z_{\text{next}})^2$$
注意，$a_{r+1}$ 在这个过程中是一个**常数**。因为 $r+1$ 这堵墙一直没有被删，所以操作 $r+1$ 左侧的元素时，计算第三项代价的左端点永远是被固定的 $a_{r+1}$。

**2. 比较“交错删除”与“不交错删除”**
假设我们采取交错删除策略，发生了两个动作：
- **动作 A**：删除 $[k+1, r]$ 中的部分元素。此时 $r+1$ 右侧的点是 $i_1$。
- **动作 B**：删除 $k$。此时 $r+1$ 右侧的点变成了 $i_2$。

这两个动作在第三项上产生的总能量收益为：
$$\text{收益}_{\text{交错}} = (a_{r+1} - z_{i_1})^2 + (a_{r+1} - z_{i_2})^2$$

如果我们**不交错**，即把动作 A 和动作 B 集中在同一个时刻（要么都在 $i_1$ 发生，要么都在 $i_2$ 发生），那么收益有两种可能：
- 都在 $i_1$ 发生：$2(a_{r+1} - z_{i_1})^2$
- 都在 $i_2$ 发生：$2(a_{r+1} - z_{i_2})^2$

根据基本不等式 $\max(2X, 2Y) \ge X + Y$（令 $X = (a_{r+1} - z_{i_1})^2$，$Y = (a_{r+1} - z_{i_2})^2$），我们立刻得出：
$$\max\Big( 2(a_{r+1} - z_{i_1})^2, \; 2(a_{r+1} - z_{i_2})^2 \Big) \ge (a_{r+1} - z_{i_1})^2 + (a_{r+1} - z_{i_2})^2$$

**结论：**
上述不等式证明了：**对于任何一种用到多个不同 $i$ 的“交错删除”方案，我们总是能找到一种只用单一最优 $i$ 的“一口气删除”方案，使得总能量更大（或至少相等）。**

因此，DP 状态设计中直接承诺 $[k+1, r]$ 和 $k$ 共享同一个右端点 $i$ 是完全严谨的。区间 DP 正是巧妙利用了这个极值性质，将复杂的“交错排列”操作压缩成了简单的连续区间合并操作，从而完美保障了正确性和时间复杂度。
