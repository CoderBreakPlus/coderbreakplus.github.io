优雅的暴力美学。**一句话题解：正着主旋律，倒着 dp 算出每条边对答案的贡献。**

### 正着主旋律

回忆主旋律的做法。$f(S)$ 表示点集 $S$ 形成强连通图的方案数，则 

+ $f(S) = \prod\limits_{e \in S} (1+w_e) - \sum \limits_{T \subseteq S} g(T)\prod\limits_{u,v \in S, v \notin T} (1+w_{u,v})$
+ $g = -\exp(-f)$ 

意思是缩点后形成 DAG，则套用 DAG 容斥的做法。

于是有这样的代码。请注意上面的式子不严谨，且上面的变量名和下面的不太一样。还有一点细节。


```cpp
n=read(),m=read();
for(int i=0;i<n;i++)
	for(int j=0;j<n;j++)
		e[i][j]=1;
for(int i=1;i<=m;i++){
	u[i]=read()-1,v[i]=read()-1,w[i]=read();
	e[u[i]][v[i]]=(ull)e[u[i]][v[i]]*(w[i]+1)%mod;
} // 每条边可选可不选，相当于乘上 (1+w[i])
for(int i=0;i<n;i++){
	h[i][0]=1;
	for(int j=1;j<(1<<n);j++){
		int x=__builtin_ctz(j);
		h[i][j]=(ull)h[i][j^(1<<x)]*e[x][i]%mod;
	}
} // 预处理一个点到一个集合的系数

g[0]=1;
for(int i=0;i<(1<<n);i++){
	for(int j=i&(i-1);j;j=(j-1)&i)if(lowbit(j)==lowbit(i)){
		g0[i]=(g0[i]+(ull)(mod-f[j])*g[i^j])%mod;
	}

	int prod=1;
	for(int j=0;j<n;j++)if((i>>j)&1)prod=(ull)prod*h[j][i]%mod;
	f[i]=prod; // 总的生成子图的方案数

	val[0]=1;
	for(int j=i;j;j=(j-1)&i){
		int k=i^j;
		if(k){
			int t=__builtin_ctz(k);
			val[k]=(ull)val[k^(1<<t)]*h[t][i]%mod;
		}
		if(j==i)
			f[i]=(f[i]+(ull)(mod-g0[j])*val[k])%mod;
		else
			f[i]=(f[i]+(ull)(mod-g[j])*val[k])%mod;
	} // 容斥部分
	addmod(g[i]=g0[i]+f[i]); // 加上单独的贡献
}
```

### 倒着推贡献

我们想知道对于每个 ``e[i][j]``，它乘以多少的系数，最终加进了 ``f[(1<<n)-1]``。

在 Gemini 的指导下，终于学会了有这么一种机械化的写法，能暴力将**所有参与运算的数对答案的贡献**都算出来。

```cpp
int f[1<<16],g[1<<16],g0[1<<16],h[16][1<<16],val[1<<16],e[16][16];
```

在倒过来 dp 之前，对于以上这些我们开过的所有辅助数组，我们都开一个对应的系数数组。

```cpp
int df[1<<16],dg[1<<16],dg0[1<<16],dh[16][1<<16],dval[1<<16],de[16][16];
```

直接对着之前正着 DP 的代码贺就好了。

具体地，初始显然有 ``df[(1<<n)-1]`` 为 $1$。然后倒过来看正着 DP 时的每一步运算，若一步运算形如：

$$A \leftarrow A + B_1 \times B_2 \times \cdots \times B_m$$

令 $A'$ 表示 $A$ 对答案的贡献系数，我们可以就此反推 $B'_1 \sim B'_m$：

$$B'_i \leftarrow B'_i + A' \times \prod \limits_{j \in [1,i) \cup (i,m]} B_j$$

这是用组合意义容易理解的。也可以看作求偏导的过程，但是作为文盲不想展开这个角度的解释。

代码中注释是转写前的运算，对应起来看就很好理解了。

```cpp
df[(1<<n)-1]=1;
for(int i=(1<<n)-1;i>=0;i--){
	// addmod(g[i]=g0[i]+f[i]);
	addmod(dg0[i]+=dg[i]);
	addmod(df[i]+=dg[i]);

	val[0]=1;
	for(int j=i&(i-1);j;j=(j-1)&i){
		int k=(i^j);
		int t=__builtin_ctz(k);
		val[k]=(ull)val[k^(1<<t)]*h[t][i]%mod;
	}
	for(int k=i;;k=(k-1)&i){
		dval[k]=0;
		if(!k)break;
	}
	for(int k=i&(i-1);;k=(k-1)&i){
		int j=i^k;
		if(j==i){
			// f[i]=(f[i]+(ull)(mod-g0[j])*val[k])%mod;
			dg0[j]=(dg0[j]+(ull)(mod-df[i])*val[k])%mod;
			dval[k]=(dval[k]+(ull)(mod-df[i])*g0[j])%mod;
		}else{
			// f[i]=(f[i]+(ull)(mod-g[j])*val[k])%mod;
			dg[j]=(dg[j]+(ull)(mod-df[i])*val[k])%mod;
			dval[k]=(dval[k]+(ull)(mod-df[i])*g[j])%mod;
		}
		if(k){
			int t=__builtin_ctz(k);
			// val[k]=(ull)val[k^(1<<t)]*h[t][i]%mod;
			dval[k^(1<<t)]=(dval[k^(1<<t)]+(ull)dval[k]*h[t][i])%mod;
			dh[t][i]=(dh[t][i]+(ull)dval[k]*val[k^(1<<t)])%mod;
		}else break;
	}

	int prod=1;
	for(int j=0;j<n;j++)if((i>>j)&1)prod=(ull)prod*h[j][i]%mod;

	// f[i]=prod;
	for(int j=0;j<n;j++)if((i>>j)&1)
		dh[j][i]=(dh[j][i]+(ull)df[i]*prod%mod*INV(h[j][i]))%mod;

	for(int j=i&(i-1);j;j=(j-1)&i)if(lowbit(j)==lowbit(i)){
		// g0[i]=(g0[i]+(ull)(mod-f[j])*g[i^j])%mod;
		df[j]=(df[j]+(ull)(mod-dg0[i])*g[i^j])%mod;
		dg[i^j]=(dg[i^j]+(ull)(mod-dg0[i])*f[j])%mod;
	}
}
```
